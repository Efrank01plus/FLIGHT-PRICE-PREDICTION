## 📌 Project Overview

This project analyzes airline ticket pricing data to identify the factors that influence flight prices. The study focuses on understanding how variables such as the number of stops, airline, departure location, destination, and flight duration affect ticket prices.

The analysis was conducted using Microsoft Excel, Pivot Tables, Pivot Charts, and descriptive statistics to uncover pricing patterns and provide business insights.

---

## 🎯 Objectives

The objectives of this project are to:

- Analyze the impact of the number of stops on ticket prices.
- Identify the airline with the highest average ticket price.
- Examine whether departure location affects ticket prices.
- Determine whether destination affects ticket prices.
- Explore factors that contribute to variations in airline ticket pricing.

---

## 📊 Research Questions

1. How does flight duration affect ticket price?
2. Does the number of stops affect ticket price?
3. Which airline has the highest average ticket price?
4. Does departure location affect ticket price?
5. Does destination affect ticket price?

## 🔍 Key Findings

- Ticket prices vary across airlines.
- The number of stops influences average ticket prices.
- Certain departure locations have higher average fares.
- Ticket prices differ significantly across destinations.
- Flight duration shows a relationship with ticket price.
